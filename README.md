# [NOSTR User Event/Replies Viewer](https://sunmoonron.github.io)

- Connects to 5 relays to fetch NOSTR events
- Enables viewing user posts and/or replies in maximum 720 hours (30 days) range
- Ccmpares two users side-by-side
- Showcases thread interactions visually
